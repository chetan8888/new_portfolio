Interactive Map for Property Booking

This mobile application is a handy tool for any broker. On the first screen it shows a list of all the projects available in the agency. On click of any project opens a detail screen that shows all the physical and geological data of the site as well as graphical statistics depicting the number of bookings, number of unbooked properties in the site as well as projected future price of the properties. A further third screen opens an interactive color coded map of the floor plans using which a broker can book a property on behalf of a customer, cancel a booking as well as get push notifications for any booking.
